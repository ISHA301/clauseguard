## 🎯 The Problem

Legal teams spend an average of **3 hours reviewing a single contract**. For teams handling 500+ contracts a year, that's 188 out of 250 working days spent solely on contract review — before a single business decision is made.

The real cost isn't the review time. It's what gets missed:

- Auto-renewals nobody wanted — locked in for another $120,000 year
- Cancellation windows missed by days — triggering $60,000 penalties
- Hidden clauses buried in page 7 — discovered only after signing
- No clear action plan — just a pile of legal text and a looming deadline

**One missed deadline can cost $60,000–$180,000+. ClauseGuard exists to prevent that.**

---

## 💡 The Solution

**ClauseGuard** is an AI-powered contract risk intelligence agent built on Microsoft Azure AI Foundry. It reads contracts from a Foundry IQ knowledge base and reasons through them in 4 structured steps — extracting clauses, detecting risks, calculating financial exposure, and recommending specific actions with exact deadlines.

No legal expertise required. No manual scanning. No missed deadlines.

---

## 🏗️ Architecture

```
User asks a question
        ↓
ClauseGuard Agent (grok-4.3 · Azure AI Foundry)
        ↓
Foundry IQ — Azure AI Search knowledge base
Retrieves grounded contract context with citations
        ↓
4-Step Reasoning Chain
    Step 1 — Clause Extraction
    Step 2 — Risk Detection (HIGH / MEDIUM / LOW)
    Step 3 — Financial Impact Analysis ($$$)
    Step 4 — Recommended Actions with deadlines
        ↓
Structured risk report — grounded in actual contract text
```

## 💬 Intelligent Response Modes

ClauseGuard uses intent-based reasoning — not keyword matching — to adapt its response format to what you actually need:

| What you ask | What you get |
|-------------|-------------|
| Full contract analysis | Complete 4-step structured report |
| Specific question | Direct 2–4 sentence answer |
| Quick overview | 3–5 bullet points under 100 words |
| What should I do? | Numbered action list with exact deadlines |

## 🚀 How to Run

### Prerequisites
- Python 3.10+
- Azure AI Foundry account
- Azure subscription

### Setup
```bash
git clone https://github.com/ISHA301/clauseguard
cd clauseguard
python -m venv .venv
.venv\Scripts\activate
pip install azure-ai-projects azure-identity openai python-dotenv
```

### Configure
Create a `.env` file:
```
AZURE_AI_PROJECT_ENDPOINT=your-endpoint-here
AZURE_AI_MODEL_DEPLOYMENT=grok-4.3
AZURE_AI_API_KEY=your-api-key-here
```

### Run
```bash
python clauseguard_agents.py
```
